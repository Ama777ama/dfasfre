# Программа поиска факториала на языке java script


## Как работает программа

***тут мы объявление переменных***

```
let factorial = 6;
let n = 1;
let count = 1;
```

***вычисление факториала***

```
for(let i = 1; i <= factorial; i++){
    n=n*i;
}
```

***вывод результата на экран***

```
console.log(n);
```
